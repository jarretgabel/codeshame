# CodeSha.me

***

### Local Setup
1. `git clone git@github.com:robojack/codeshame.git`
1. `$ cd path/to/project`
1. `$ bundle install`
1. `$ npm install`
1. `$ npm install -g grunt-cli` (Assuming you don't already have it)
1. `$ foreman start`
1. Point your browser to [http://localhost:4000](http://localhost:4000)
